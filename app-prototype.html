核心变量
--bg页面基础背景色

--s1/s2/s3表面层级，越大越亮
--card卡片专用背景色
--text/2/3文字三级层次
--accent主强调色（紫）
--accent-s主色半透明背景
--accent-g主色较深透明边框
--teal成功色（青绿）
--orange警告 / 强调色
--red危险 / 错误色
--border/2分隔线 / 控件边框
主题切换模式
:root { /* 深色默认 */ --accent: #8B72FF; --bg: #07070E; } @media(prefers-color-scheme:light) { :root { --accent: #6B52F0; --bg: #F2F2F7; } }
